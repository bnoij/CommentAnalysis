## 🚀 如何开始使用

### 方式 1: 自动启动 (推荐)

```bash
# Windows
start.bat

# Mac/Linux
chmod +x start.sh
./start.sh
```

### 方式 2: 手动启动

```bash
# 后端
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux
pip install -r requirements.txt
python app.py

# 前端 (新终端)
npm install
npm run dev
```

### 方式 3: 测试后端

```bash
cd backend
python test.py
```
